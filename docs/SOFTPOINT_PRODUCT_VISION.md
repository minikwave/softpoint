# SoftPoint 제품 비전

**SoftPoint**는 다른 프로덕트에 **보상·인센티브 레이어**로 붙는 프로그래머블 크레딧 엔진입니다. 지갑·거래소가 아니라, 발행·적립·사용·교환·정산을 API와 영수증으로 추적합니다.

## 핵심 가치

1. **임베드형 보상** — 게임·커머스·핀테크·콘텐츠 앱에 SP 지급 API만 연결
2. **한 흐름** — 모으기(결제·Walk·광고·매장) → 쓰기(결제·마켓) → 바꾸기(기프티콘·스테이블)
3. **운영 가능** — 정책·예외·감사·준비금·영수증 SSOT

## 기능 축 (로드맵 포함)

| 축 | v1 데모 | 다음 단계 |
|----|---------|-----------|
| **적립** | 결제 캐시백, 활동 카탈로그(Walk/광고/매장) | 실 걸음·광고 SDK, 파트너 웹훅 |
| **사용** | SP 결제, 기프티콘 교환 | 포인트 마켓 P2P·재고 |
| **정산** | 스테이블 전환 요청·운영 승인 | 온체인 어댑터, 자동 대사 |
| **연동** | REST + 온보딩 문서 | 샌드박스 키, 파트너 포털 |

## 브랜드·용어

- 제품명: **SoftPoint** (한·영 동일)
- 포인트 단위: **SP**
- API 경로 `/v1/paypoint/*` 는 안정적 통합 surface (내부 레거시 네이밍)
