# 에이전트·협업 메모

- **제품명**: **SoftPoint** (영문·슬러그 **softpoint**). 레포: `minikwave/softpoint`.
- **구성**: Engine API · Consumer D-App · Operator Console · `@softpoint/domain` · `@softpoint/sdk`.
- **레거시**: `pay-saum` / `paypoint` 레포·`쌓음(saum)` 브랜드는 이전 세대. 신규 작업은 **softpoint만** 대상.
- **`moum` / 「모음」** 은 공식 제품명·패키지명·브랜드로 쓰지 않는다.
- **SoftCredit** 제품명 금지. SoftPG **agent credit** 경로에 SoftPoint를 끼우지 않는다.
- SoftPay 연계: `/hooks/softpay` · [docs/SOFT_STACK_BOUNDARY.md](./docs/SOFT_STACK_BOUNDARY.md)

자세한 표: [docs/PRODUCT_NAMING.md](./docs/PRODUCT_NAMING.md)
